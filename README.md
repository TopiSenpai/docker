# docker
this is my docker-compose configuration for my Virtual Private Server :)

### Container Starting Order:

    1. database
    
    2. portainer
    
    3. teamcity
    
    4. kittybot
        
    5. item-warehouse
    
    6. minecraft
    
    7. nginx

or run the start script

### Updating KittyBot
run `$ ./docker/kittybot/update`

### Updating Item-Warehouse
run `$ ./docker/item-warehouse/update`
